Input Mask Directive

Use pre-defined mask literals to define a mask. Any character not recognized as a mask literal will be added as a constant.

Mask literals:

  • A - Place holder for any alpha character
  • 9 - Place holder for any numeric character
  • * - Place holder for any alpha numeric character
